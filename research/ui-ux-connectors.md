# 🎨 Eric's Core UI/UX Connectors

Catatan ini melacak 5 konektor utama dalam ekosistem desain-ke-kode (design-to-code) yang diintegrasikan untuk kebutuhan pengembangan frontend otonom.

## 📐 Design & Prototyping Platforms
* **[[figma]]**: Platform utama untuk kolaborasi desain UI/UX dan penarikan aset komponen.
* **[[penpot]]**: Platform desain open-source berbasis web untuk alternatif referensi layouting agen.
* **[[ui ux promax]]**: Standardisasi dan aset desain tingkat lanjut untuk kualitas visual maksimal.

## 💻 Frontend & Component Libraries
* **[[21st dev]]**: Hub/repositori komponen AI untuk mempercepat implementasi komponen siap pakai.
* **[[framer motion]]**: Library animasi untuk menghidupkan komponen UI dengan transisi tingkat tinggi.
